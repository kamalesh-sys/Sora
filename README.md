# Sora Expense

Track it. Know it.

A simple Android app for house expenses, bills, budgets, and reports.

## Download

Get the latest APK from the download page:

```text
docs/index.html
```

## Stack

- Django REST API
- PostgreSQL
- Expo React Native
- Android APK

## Notes

- Backend runs on Render.
- App talks to the Django API only.
- Secrets stay in environment variables.
